# passo-holding

**Temporary holding page — to be replaced by the Passo platform.**

A single `index.html` deployed to `passo.ad` while the platform is in development. No framework, no build step — deploy the file directly.

## Purpose

Confirms the domain is live, communicates "coming soon", and directs enquiries to `hello@passo.ad`.

## Deployment

Drop `index.html` into any static host (Vercel, Netlify, Cloudflare Pages) pointed at `passo.ad`. No configuration needed.

## When to retire this

Replace entirely once the Passo merchant platform is ready to go live. Do not extend or build on this page.
