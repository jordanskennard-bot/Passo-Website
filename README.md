# passo-holding

Confirms the domain is live, communicates "coming soon", and directs enquiries to `hello@passoagency.com`.

Drop `index.html` into any static host (Vercel, Netlify, Cloudflare Pages) pointed at `passoagency.com`. No configuration needed.

## When to retire this

Replace entirely once the Passo merchant platform is ready to go live. Do not extend or build on this page.
